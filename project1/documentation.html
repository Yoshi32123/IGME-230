Documentation

Overall Process:

Initial Design

  • Trying to figure out how to "fill" bloks was the first milestone. Using css classes, it made it very easy to implement.
  • Implementing a "dragging" interface seemed too much work since the bloks were not pictures. Instead, it was changed to a move and place.

Errors that were overcame

  • When moving the blok right or down, it would delete a portion of the blok. To fix this, a placement array was made before implementing the move.
  • When moving to edge of the screen, it would cause indexing errors. To fix this, a min/max index was created/set based on the selected blok.
  • Hovering over a filled cell with a selected cell would delete the filled cell. To fix this, a "filled and selected" class was made.
