Proposal

Game Name:

Blok.

Concept:

A click and move block puzzle game made using DOM.

Genre:

This is an puzzle based click and move.

Platform:

Desktop only.

Story:

No story, just need to keep going as long as possible.

Gameplay:

The player will have to choose "blocks" from 3 choices by clicking, and then moving them in the grid box. If they fill an entire row or column, it will be emptied to make room for more blocks. If the can't fit any of the blocks below, it will be game over.

Controls:

  • Click a box to select the blok.
  • Arrow keys to move that blok in the main grid.
  • Enter to place the blok.
